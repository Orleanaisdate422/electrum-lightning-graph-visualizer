## 🧭 What this does

electrum-lightning-graph-visualizer is an Electrum plugin that helps you view the Lightning Network graph and payment routes in a clear way. It is made for people who use Electrum on Windows and want a simple visual view of how Lightning nodes connect.

This tool can help you:
- See the Lightning network graph in a visual format
- Review possible payment routes
- Better understand how Lightning connections flow
- Use it as an Electrum plugin inside your wallet setup

## ❓ Common questions

### Does this work without Electrum?
No. This is a plugin for Electrum, so you need Electrum installed first.

### Can I use it on Windows?
Yes. This README is set up for Windows users.

### Do I need coding knowledge?
No. You only need to download the files, place them in the right folder, and turn the plugin on in Electrum.

### What if the graph does not load?
Check that Electrum is running, your wallet is open, and your internet connection works. Then wait a bit and try again.

### Can I use it with any wallet?
It is made for Electrum wallets, not every wallet app.
